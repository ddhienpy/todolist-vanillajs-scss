# TODO-List
Technologies: VanillaJS, sass

## Overview
Create TODO List using HTML, VanillaJS, sass

## Project setup
```
npm install
```

### Compiles Sass To Css
```
npm run compileSass
```

### Features
+ List of tasks
+ Add task
+ Update task
+ Delete task
+ Filter task
+ Change task status
